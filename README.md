# 프랩 애플리케이션

이 애플리케이션은 [Expo](https://expo.dev) 프로젝트로 [`create-expo-app`](https://www.npmjs.com/package/create-expo-app)로 생성되었습니다.

## 시작하기

1. 의존성 설치

   ```bash
   npm install
   ```

2. 애플리케이션 실행

   ```bash
   npm run start
   ```

   Android

   ```bash
   npm run android
   ```

   iOS

   ```bash
   npm run ios
   ```

   Web

   ```bash
   npm run web
   ```
